# Boomy Changelog

# v0.5.0-rc1

HamBuild DTA export with full move feature support, experimental move attributes (footik commands), cross-platform HamBuild execution, camera type expansion, beat/measure shifting for HamBuild compatibility as well as user preferences with last opened projects and other bug fixes like missing venues and cameras.

# v0.3.0

Added metadata editing, pre-build tests, mogg making, art cover making, xbox package creator, importing/exporting songs, new song creation (with template)
Introduced BoomyExporter

This release should be not used with milo-move-library in version 1 as well as previous boomy projects with version 1.
